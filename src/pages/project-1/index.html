Sistema de gerenciamento financeiro familiar

Aplicação desktop desenvolvida em grupo para o projeto integrador do programa Jovem Programador.

    Funcionalidades:

  • Login/Cadastro
  • Gerenciar categorias de gastos
  • Gerenciar membros da família
  • Gerenciar gastos

    Tecnologias usadas:

  • C# (.Net)
  • WindowsForms
  • Arquitetura MVC
  • MySql
  • Git | GitHub
